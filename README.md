
# About Groovy

[![Product Name Screen Shot][product-screenshot]](https://example.com)

Groovy is a sleek and intuitive offline music player app designed to provide an exceptional music playback experience right on your Android device. Seamlessly organize and play your favorite tracks with ease, whether you're on the go or relaxing at home.

### Key Features:
* Latest material 3 Design.
* Dynamic theme support.
* Native android app.

### Build With:
* Native Kotlin
* Jetpack Compose
* Media3 With Exoplayer
* Foreground Services & Runtime Permissions Enabled

### Compatibility:
* Android Version 7.0 (Nougat) or higher

Note: To get the updated code, Please follow the version number in the branch.
